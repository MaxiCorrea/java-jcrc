# java-jcrc

### Descripción
Este proyecto pretende ser una utilidad para la construcción y organización de tarjetas CRC.

### Caracteristicas
1 - Crear/Modificar/Eliminar tarjetas CRC de forma interactiva.

2 - Exportar en formato JSON y PDF.

### Tecnologías
* JavaSE 8
* swing
* swingx 1.6.1
* jasper report 6.1.0
* json-simple 1.1.1
* Mockito 1.0.3
* JUnit 4.11
